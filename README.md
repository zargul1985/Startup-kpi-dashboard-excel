# 📈 Startup KPI Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive Startup KPI Dashboard developed using Microsoft Excel. It provides a clear overview of key business performance indicators, enabling startup founders and stakeholders to monitor growth, financial performance, and operational efficiency.

## 🎯 Business Objective

The objective of this dashboard is to:

* Monitor key startup performance metrics.
* Track revenue and business growth.
* Analyze customer acquisition and retention.
* Evaluate financial performance through interactive reports.
* Support strategic business decisions with data-driven insights.

## 🛠 Tools & Features Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Formulas & Functions
* Interactive Dashboard Design

## 📂 Dataset

The dataset contains startup business information, including key performance indicators such as revenue, expenses, customer metrics, and operational data.

## 📊 Dashboard Features

* Revenue Overview
* Profit Analysis
* Customer Growth Tracking
* KPI Cards
* Interactive Filters (Slicers)
* Trend Analysis
* Dynamic Charts

## 💡 Key Insights

* Visualized business growth using interactive charts.
* Tracked important startup KPIs in a single dashboard.
* Improved decision-making through clear performance reporting.
* Designed an easy-to-use and interactive reporting interface.
